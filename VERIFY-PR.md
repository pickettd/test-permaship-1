# PR Verification Checklist

This document provides a checklist for verifying pull requests in this repository.

## Review Steps

- [ ] Code changes are clear and well-described in the PR summary
- [ ] Changes are scoped to the stated purpose of the PR
- [ ] No unintended files are included in the commit
- [ ] Branch is up to date with main before merging
